# monetize-advisor

## 概要
日本語SaaSプロダクトのマネタイズ戦略を提案するスキル。
プロダクト概要を渡すと、フリーミアム設計・価格帯・
B2Bアップセル・アフィリ提携候補を日本市場向けに提案する。

## いつ使うか
以下のいずれかをユーザーが言ったとき：
- 「マネタイズ方法を考えて」「収益化の方法を教えて」
- 「料金プランを設計したい」
- 「フリーミアムをどう設計すればいい？」
- 「B2Bプランを作りたい」
- 「このプロダクトでどう稼ぐか考えて」

## 入力
以下の情報をユーザーから受け取る。不足している場合は質問して埋める。

- **プロダクト名・概要**：何をするサービスか
- **メインターゲット**：個人 / 中小企業 / 大企業 など
- **現状**：ユーザー数・MAU・収益（任意）
- **コスト構造**：API費用・インフラ費用など（任意）
- **競合**：類似サービスと現在の価格帯（任意）

## 生成手順

1. プロダクトの特性を判断する
   （個人向け / BtoB / ツール系 / コンテンツ系）
2. 日本市場の価格感覚に合わせて設計する
3. 以下のセクションを順に出力する

## 注意事項
- 日本市場の価格感覚を基準にする
  （個人向け¥500〜¥2,000・SMB向け¥5,000〜¥30,000）
- 海外SaaSの価格をそのまま円換算で提案しない
- コスト情報がない場合は保守的に見積もる
- 確認できないデータに基づく断言はしない
